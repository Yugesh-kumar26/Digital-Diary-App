Digital Diary App
RISE Internship - Project 5

A secure and private digital space for students and individuals to record personal thoughts, ideas, and daily plans. This app focuses on data privacy and modern design for personal reflections.

Key Features
Secure Access: Protected by a 4-digit PIN (Default: 1234) to ensure a private and secure space for users.

Full CRUD Functionality: Users can Add, View, Edit, and Delete diary entries easily.

Calendar Integration: Features a calendar view to browse, select, and organize entries by specific dates.

Local Persistence: Designed to store daily notes and reflections locally on the device.

Modern UI: A clean, Teal-themed interface built with Flutter's Material 3 design system.

Technical Specifications
Framework: Flutter (Material 3)

Language: Dart

Date Formatting: intl package

Calendar Widget: table_calendar
